## 🛠️ 工具组件

### 核心组件
- **gui_main.py** - GUI主程序，包含完整界面
- **game_extractors.py** - 游戏文本提取引擎
- **core/translator.py** - 主翻译工作流
- **cli.py** - 命令行界面

### 解析器 (parsers/)
- **json_parser.py** - JSON/JSON5格式
- **csv_parser.py** - CSV/TSV表格
- **xml_parser.py** - XML格式
- **yaml_parser.py** - YAML格式
- **excel_parser.py** - Excel文件
- **binary_parser.py** - 自定义二进制

### 翻译引擎 (translators/)
- **google_translator.py** - Google翻译（免费）
- **deepl_translator.py** - DeepL翻译（高质量）
- **baidu_translator.py** - 百度翻译（中文优化）
- **local_translator.py** - 本地AI模型（离线）

## 🔧 配置文件 (config.yaml)

```yaml
# 翻译设置

  engine: "google"              # 默认引擎
  source_lang: "auto"           # 自动检测源语言
  target_lang: "zh-CN"          # 目标语言: 简体中文
  delay_between_requests: 0.3   # 请求延迟(秒)

# GUI设置
gui:
  default_width: 1200           # 窗口宽度
  default_height: 800           # 窗口高度
  auto_save: true               # 自动保存
  theme: "default"              # 主题

# 游戏检测
  supported_engines:            # 支持的引擎
    - "rpgmv"
    - "rpgvxace"
    - "wolf"
    - "unity"
    - "renpy"
    - "generic"
```

## 📌 注意事项

1. **备份重要**: 工具会自动备份，但重要游戏建议手动备份
2. **API限制**: Google翻译有频率限制，大量翻译建议添加延迟
3. **编码问题**: 工具自动检测编码，极少数情况需要手动指定
4. **加密游戏**: 部分游戏加密无法直接提取，需要先解密
5. **测试验证**: 翻译后务必测试游戏，确保格式正确

## 🆘 故障排除

**问题1: GUI无法启动**
```
解决方案: 
1. 确保PyQt5已安装: pip install PyQt5
2. 运行测试脚本: python test_gui_import.py
3. 检查错误日志
```

**问题2: 提取文本失败**
```
解决方案:
1. 确认选择的是游戏主目录
2. 检查游戏是否加密
3. 尝试使用命令行版本分析
```

**问题3: 翻译失败**
```
解决方案:
1. 检查网络连接
2. 验证API密钥
3. 增加请求延迟(delay)
4. 切换到本地引擎
```

**问题4: 编码错误**
```
解决方案:
1. 工具自动检测编码
2. 在config.yaml中手动指定
3. 使用chardet库检测
```

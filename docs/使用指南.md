# RPG Game Translator 使用指南

## 快速开始

### 1. 安装依赖

```bash
cd rpg_translator
pip install -r requirements.txt
```

### 2. 基本使用

#### 翻译单个文件

```bash
# 使用 Google 翻译（免费，无需配置）
python cli.py translate dialogue.json --target zh-CN

# 指定输出路径
python cli.py translate dialogue.json -o translated/dialogue_zh.json

# 指定源语言
python cli.py translate dialogue.json --source en --target zh-CN
```

#### 批量翻译目录

```bash
# 翻译整个目录
python cli.py batch ./game_data --recursive

# 只翻译特定格式
python cli.py batch ./game_data --extensions .json .csv

# 指定输出目录
python cli.py batch ./game_data -o ./translated
```

## 配置说明

### 方法一：使用配置文件（推荐）

#### 1. 创建配置文件

```bash
# 生成默认配置文件
python cli.py config --init
```

这会在当前目录创建 `config.yaml` 文件。

#### 2. 编辑配置文件

打开 `config.yaml`，根据需要修改：

```yaml
# 翻译设置
translation:
  source_lang: "auto"      # 源语言，auto 表示自动检测
  target_lang: "zh-CN"     # 目标语言
  engine: "google"         # 翻译引擎
  
  # 批量设置
  batch_size: 50           # 每批翻译文本数
  delay_between_requests: 0.5  # API 请求间隔（秒）

# API 密钥配置
apis:
  deepl:
    api_key: "your-deepl-api-key-here"
  baidu:
    app_id: "your-baidu-app-id"
    secret_key: "your-baidu-secret-key"
```

#### 3. 查看当前配置

```bash
python cli.py config --show
```

---

### 方法二：使用环境变量

对于敏感信息（API 密钥），建议使用环境变量：

#### 1. 创建 .env 文件

```bash
# 复制示例文件
cp .env.example .env
```

#### 2. 编辑 .env 文件

```env
# DeepL API（可选）
DEEPL_API_KEY=your_deepl_api_key_here

# 百度翻译 API（可选）
BAIDU_APP_ID=your_baidu_app_id
BAIDU_SECRET_KEY=your_baidu_secret_key

# OpenAI API（用于高级本地翻译，可选）
OPENAI_API_KEY=your_openai_api_key
```

#### 3. 加载环境变量

工具会自动读取 `.env` 文件，无需额外操作。

---

### 方法三：命令行参数

直接在命令行中指定配置，优先级最高：

```bash
python cli.py translate input.json \
    --engine deepl \
    --source en \
    --target zh-CN \
    --delay 1.0 \
    --output-dir ./output
```

## 各翻译引擎配置

### Google Translate（推荐新手）

**特点：**
- ✅ 免费，无需 API 密钥
- ✅ 支持多种语言
- ⚠️ 可能有请求限制

**配置：**
```bash
# 无需配置，直接使用
python cli.py translate input.json --engine google
```

---

### DeepL（推荐专业翻译）

**特点：**
- ✅ 翻译质量高
- ✅ 免费版：每月 50 万字符
- ⚠️ 需要 API 密钥

**获取 API 密钥：**
1. 访问 https://www.deepl.com/pro-api
2. 注册账号
3. 获取 API Key

**配置：**

方法一：环境变量
```env
DEEPL_API_KEY=your-api-key-here
```

方法二：配置文件
```yaml
apis:
  deepl:
    api_key: "your-api-key-here"
    free_mode: true  # 使用免费版
```

**使用：**
```bash
python cli.py translate input.json --engine deepl
```

---

### 百度翻译

**特点：**
- ✅ 免费版：每月 200 万字符
- ✅ 支持中日韩语言
- ⚠️ 需要 App ID 和密钥

**获取 API 凭证：**
1. 访问 https://fanyi-api.baidu.com/
2. 注册并创建应用
3. 获取 APP ID 和密钥

**配置：**

方法一：环境变量
```env
BAIDU_APP_ID=your-app-id
BAIDU_SECRET_KEY=your-secret-key
```

方法二：配置文件
```yaml
apis:
  baidu:
    app_id: "your-app-id"
    secret_key: "your-secret-key"
```

**使用：**
```bash
python cli.py translate input.json --engine baidu
```

---

### 本地 AI 模型

**特点：**
- ✅ 完全离线运行
- ✅ 无请求限制
- ⚠️ 需要下载模型（约 300MB）
- ⚠️ 首次运行较慢

**安装依赖：**
```bash
pip install transformers torch
```

**配置：**
```yaml
local_model:
  enabled: true
  model_name: "Helsinki-NLP/opus-mt-en-zh"
  device: "auto"  # 自动选择 CPU 或 GPU
  cache_dir: "./models"
```

**使用：**
```bash
python cli.py translate input.json --engine local --source en --target zh
```

**可用模型：**
- `Helsinki-NLP/opus-mt-en-zh` - 英→中
- `Helsinki-NLP/opus-mt-en-ja` - 英→日
- `Helsinki-NLP/opus-mt-en-ko` - 英→韩
- `Helsinki-NLP/opus-mt-zh-en` - 中→英

## 高级配置

### 文本处理规则

在 `config.yaml` 中配置：

```yaml
processing:
  # 最小/最大文本长度
  min_text_length: 2
  max_text_length: 5000
  
  # 跳过模式（正则表达式）
  skip_patterns:
    - "^[0-9]+$"           # 纯数字
    - "^[a-zA-Z0-9_]+$"    # 变量名
  
  # 保留模式（不翻译占位符）
  preserve_patterns:
    - "\\{[^}]+\\}"        # {variable}
    - "<[^>]+>"            # <tag>
```

### 输出设置

```yaml
output:
  directory: "./output"    # 输出目录
  backup: true             # 创建备份
  log_file: "translation_log.json"  # 翻译日志
```

### 格式特定设置

```yaml
formats:
  json:
    encoding: "utf-8"
    indent: 2
  
  csv:
    delimiter: ","
    text_columns: [1, 2, 3]  # 指定要翻译的列
  
  excel:
    sheet_index: 0          # 工作表索引
    text_columns: ['A', 'B'] # 指定列
```

## 常见问题

### Q: 如何查看支持的格式和语言？

```bash
python cli.py info
```

### Q: 翻译速度慢怎么办？

1. 减少请求延迟：`--delay 0.3`
2. 使用本地模型：`--engine local`
3. 增加批量大小：`--batch-size 100`

### Q: 如何只翻译特定内容？

使用 CSV/Excel 时指定列：
```yaml
formats:
  csv:
    text_columns: [1, 3, 5]  # 只翻译第2、4、6列
```

### Q: 翻译失败怎么办？

1. 检查网络连接
2. 验证 API 密钥是否正确
3. 查看翻译日志 `translation_log.json`
4. 尝试增加重试次数：
```yaml
translation:
  max_retries: 5
  retry_delay: 3.0
```

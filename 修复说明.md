### 🔧 修复4: BaseTranslator抛出异常而非静默失败
**问题**: translate_with_retry返回原文，导致无法捕获错误

**修复**: 在 `translators/base_translator.py` 中抛出RuntimeError

```python
def translate_with_retry(self, text: str) -> str:
    for attempt in range(self.max_retries):
        try:
            return self.translate(text)
        except Exception as e:
            last_error = e
            if attempt < self.max_retries - 1:
                time.sleep(self.retry_delay * (attempt + 1))
    
    # 所有重试失败，抛出异常！
    raise RuntimeError(f"Translation failed after {self.max_retries} attempts. Last error: {last_error}")
```

**状态**: ✅ 完成并验证通过

### 🔧 修复5: GUI接入Parser系统（最高优先级）
**问题**: GUI使用json.load()暴力解析，破坏嵌套结构

**修复**: 重写 `gui_main.py` 的load和save方法

```python
def load_file_for_translation(self, item):
    # 使用Parser系统
    from parsers import get_parser
    parser = get_parser(str(file_path))
    segments = parser.parse()
    
    # 使用location作为key（精准定位）
    for segment in segments:
        self.text_data.append({
            'key': segment.location,  # 精准路径！
            'original': segment.text,
            'translated': segment.translated_text or ''
        })

def save_file(self):
    # 使用Parser.reconstruct()保持嵌套结构
    segments = []
    for item in self.text_data:
        segment = TextSegment(
            text=item['original'],
            location=item['key'],
            translated_text=item['translated'] or item['original']
        )
        segments.append(segment)
    
    reconstructed = self.current_parser.reconstruct(segments)
    self.current_parser.save(reconstructed, str(self.current_file))
```

**状态**: ✅ 完成
